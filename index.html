경조사비 가계부 – 지원 페이지

1. 앱 소개

경조사비 가계부(EventExpenses) 앱은 결혼식, 장례식, 돌잔치 등 각종 경조사에서 발생하는 지출과 수입을 쉽고 빠르게 기록하고 관리할 수 있는 앱입니다.

  • 직관적인 인터페이스로 누구나 쉽게 사용 가능
  • 모든 데이터는 사용자의 기기에만 저장되며 외부 서버로 전송되지 않음

2. 주요 기능

  • 간편 입력: 이름, 관계, 행사명, 날짜, 금액 등을 빠르게 입력
  • 검색 기능: 이름, 관계, 행사명 기준으로 받은/보낸 금액 확인
  • 월별 통계: 월별 지출·수입 확인, 경조사비 흐름 파악
  • 엑셀 내보내기/불러오기: 백업 및 복원 가능
  • 데이터 안전: 개인정보는 기기 내에서만 관리

3. FAQ (자주 묻는 질문)

Q1. 데이터는 어디에 저장되나요?
모든 데이터는 사용자의 기기(Local Storage)에 저장됩니다. 앱 삭제 시 모든 데이터도 삭제됩니다.
Q2. 데이터 백업은 어떻게 하나요?
앱 내 엑셀 내보내기 기능을 이용해 XLSX 파일로 백업 가능합니다.
Q3. 다른 기기에서 데이터를 불러오려면?
백업한 XLSX 파일을 다른 기기로 옮긴 뒤 엑셀 불러오기 기능을 사용하면 됩니다.
Q4. 개인정보를 수집하나요?
앱은 개인정보를 수집하거나 외부로 전송하지 않습니다.

5. 개인정보 처리방침 (Privacy Policy)

경조사비 가계부(EventExpenses) 앱은 사용자의 개인정보를 수집하지 않으며, 모든 데이터는 기기 내부에만 저장됩니다.

수집하는 개인정보

  • 이름, 연락처, 주소 등 신원 확인 정보 없음
  • 위치, 결제 정보, 광고 식별자(IDFA) 등 없음

데이터 저장 방식

  • 데이터는 사용자의 기기에만 저장
  • 개발자는 데이터 접근 불가
  • 앱 삭제 시 데이터 모두 삭제

데이터 백업 및 복원

  • 엑셀(XLSX) 파일로 백업 가능
  • 사용자가 직접 지정한 경로에만 저장/복원 가능

제3자 제공

  • 현재 광고/분석 도구 없음 → 제3자 제공 없음
  • 향후 광고/분석 추가 시 내용 업데이트 필요

권한 요청

  • 파일 접근 권한만 요청 (엑셀 내보내기/불러오기 용)
  • 그 외 권한 요청 없음

아동 보호

  • 아동 개인정보 수집 없음
